## Awaiting-deploy verification plan

`pnpm install`, `pnpm build:server`, `sudo systemctl restart nomos-cp nomos-pdp`
(or pm2 restart all), re-run:

```bash
NOMOS_API_KEY='<dogfood key>' NOMOS_ORG_ID='ed539890-…' GITHUB_TEST_REPO='admin-brickexchange/smoke-test-01' \
  pnpm tsx scripts/dogfood-tier1.mts
```

Expected after deploy:
- `google_list_files (read)` → `upstream=200` (PDP fix #2 lands)
- A re-run of `/v1/agent/me/tools` against the dogfood key returns
  `integrations: ["github","google","notion","slack"]` (CP fix #3 lands)

And separately re-run the auto-approve poller against
`scripts/simulate-approval.mts` on the VM — that path will be unblocked
once the stale-import fix (#1) reaches the deployed checkout.
