Nomos brokers two kinds of providers:

- **OAuth providers** — you do a one-time consent flow in the dashboard; the broker
  encrypts and stores the token; agents get short-lived UCANs scoped to one command at a
  time.
- **Non-OAuth providers** (filesystem, SSH, cloud) — the PDP holds keys / credentials
  via environment or federated identity; scoping happens through Cedar policy +
  UCAN constraint.
